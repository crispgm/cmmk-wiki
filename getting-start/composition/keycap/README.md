# 键帽

键帽是客制化一大常见玩法，由于键帽会覆盖键盘外表大部分区域，所以键帽的不同很大程度影响键盘整体的整体外观。

键帽也是抄袭和侵权的重灾区。

## 键帽种类

不同种类的键帽高度和形状不同，除外观的明显区别外，实际的手感和声音也会不同。
